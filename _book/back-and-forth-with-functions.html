10 Back and forth with functions

Differentiation and anti-differentiation are the back and forth of a tennis game

Objectives: 

i. (Int-1a) Understand anti-differentiation as the reverse process of differentiation 
#. (Int-1b) Utilize the following basic rules of anti-differentiation to find the anti-derivatives of the basic modeling functions (MMAC pgs. 632-633): 
    a. Constant Rule 
    b. Power Rule 
    c. Exponential Rule 
    d. Sinusoidal Rule 
    e. Logarithm Rule (i.e. anti-derivative of )

  1. Finding Anti-derivatives

    Objectives: 

    1. (Int-2a) Utilize Euler’s Method to approximate anti-derivatives when the value of  is small 
    2. (Int-2b) Utilize your knowledge of derivatives rules to backwards solve for anti-derivatives 
    3. (Int-2c) Understand the notation of an indefinite integral to include
      1. the integrand symbol 
      2. variable of integration (“with respect to” variable)
      3. the constant of integration 
    4. (Int-2d) Understand the relationship between a (child) function and its (parent) integral
    5. (Int-2e) Determine the units of an anti-derivative given the base function and the variable of integration  
    6. (Int-2f) Understand that the output of anti-differentiation is a function known as a general solution and more information is necessary to find the particular solution 

  2. The Definite Integral

    Readings: 614-616

    Objectives: 

    1. (Int-3a) Understand the notation of limits of integration within a definite integral 
    2. (Int-3b) Determine to units of a definite integral (MMAC pg. 614) 
    3. (Int-3c) Use the algebraic properties of definite integrals (MMAC pg. 615-616) to calculate definite integrals 
    4. (Int-3d) Calculate definite integrals graphically 

    Additional Resources: MMAC Section 6.2 

  3. The Fundamental Theorem of Calculus

    Objectives: 

    1. (Int-4a) Understand the Fundamental Theorem of Calculus Part 2 as how to evaluate definite integrals 
    2. (Int-4b) Comprehend how the Fundamental Theorem of Calculus Part 2 graphically can calculate the area between curves (MMAC pgs. 644-645) 
    3. (Int-4c) Understand the Fundamental Theorem of Calculus Part 1 as the net accumulation function 
    4. (Int-4d) Understand how a definite integral with a variable limit of integration outputs a function 

    Additional Resources: MMAC Section 6.3 & 6.4 only pgs. 640-645  

  4. Understanding Anti-derivatives

    Objectives: 

    1. (Int-5a) Determine dimensions and units utilizing a graph 
    2. (Int-5b) Graph the antiderivative of a function given the graph of the base function and a single point on the anti-derivative 
    3. (Int-5c) Know common scientific application relationships between base functions and anti-derivatives 
      1. Acceleration, Velocity, Position 
      2. Force, Work 
      3. Area, Volume 

  5. Further applications of Anti-derivatives

    Objectives: 

    1. (Int-6a) Understand the relationship between the hump function and the sigmoidal function 
    2. (Int-6b) Discuss the scientific application of probability density and cumulative probability 
    3. (Int-6c) Understand the concept of expected value 

  6. Algebraic Integration

    Objectives: 

    1. (Int-7a) Find the integral algebraically using the technique u-substitution  
    2. (Int-7b) Find the integral algebraically of select functions using Integration by Parts 

  7. Multivariate Integration

    Objectives: 

    1. (Int-8a) Conduct anti-differentiation when multiple variables are in the base function 
    2. (Int-8b) Conduct multiple iterations of anti-differentiation when multiple variables are in the base function in the correct order 
    3. (Int-8c) Understand the dimensions and units when integrating with respect to multiple variables
