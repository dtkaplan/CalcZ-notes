The Second Fundamental Theorem of Calculus says that if \(f(x)\) is a smooth function, you can calculate \(\int_a^b f(x) dx\) by finding an anti-derivative \(F(x)\) and evaluating it at the bounds of integration as \(F(b) - F(a)\).

Using the Second Fundamental Theorem, compute the value of the definite integral \[\int_{-2}^4 \frac{1}{x^2} dx\] This looks perfectly ordinary, but it is wrong. The antiD()-constructed anti-derivative gets it right:

F <- antiD(1/x^2 ~ x)
F(4) - F(-2)
