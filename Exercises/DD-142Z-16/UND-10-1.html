UND 1.10

Consider the following discrete-time dynamics: \[x_{t+1} = \alpha x_t \exp(-\beta x_t^3)\]

In the sandbox, implement the dynamical function and call it simply f. You’re going to use this function in answering the later functions.

f <- makeFun( ..your.stuff.. ~ x, alpha = 2.72, beta=0.33)
slice_plot(f(x) ~ x, domain(x=c(0,2))) %>%
  gf_abline(yintercept=0, slope=1, color="blue") %>%
  gf_refine(coord_fixed())

Note that you should write alpha in your function for the symbol \(\alpha\), and beta for \(\beta\). The two arguments at the end of the definition set default values for these two parameters. Also note that the dynamical function have been written in terms of \(x\), not \(x_t\). Of course, it doesn’t really matter what you choose for the name of the argument to a function, so long as you use that name consistently.
